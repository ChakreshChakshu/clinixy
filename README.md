# 🩺 Clinixy – Doctor Appointment Booking App

Clinixy is a full-stack healthcare platform built with Next.js that allows patients to easily find doctors, check availability, and book appointments online. It also provides role-based dashboards for patients, doctors, and admins to manage appointments and medical data efficiently.

---

## 🚀 Features

* 🔍 Search doctors by specialization
* 📅 Real-time appointment booking
* 🔐 Secure authentication & authorization
* 👨‍⚕️ Role-based dashboards (Patient / Doctor / Admin)
* 💊 Digital prescriptions
* 📹 Video consultation (planned / integrated)
* 💳 Online payments (planned / integrated)

---

## 🛠 Tech Stack

* **Frontend:** Next.js (App Router), React
* **Backend:** Node.js, Express
* **Database:** MongoDB
* **Authentication:** JWT / NextAuth
* **Deployment:** Vercel

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/clinixy.git
cd clinixy
```

### 2. Install dependencies

```bash
npm install
```

### 3. Setup environment variables

Create a `.env.local` file and add:

```
MONGODB_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

### 4. Run the development server

```bash
npm run dev
```
